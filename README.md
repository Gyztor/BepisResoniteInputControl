# ResoniteInputControl

[![Thunderstore Badge](https://modding.resonite.net/assets/available-on-thunderstore.svg)](https://thunderstore.io/c/resonite/)

A [Resonite](https://resonite.com/) mod that allows you to use Variables to toggle whether a controller action (i.e. Jump, Move and Turn) continue to function based on a variable in your User Root.

Current Variables:
* User/InputControl.Left.Move
* User/InputControl.Left.Turn
* User/InputControl.Left.Jump
* User/InputControl.Right.Move
* User/InputControl.Right.Turn
* User/InputControl.Right.Jump

## Installation (Manual)
1. Install [BepisLoader](https://github.com/ResoniteModding/BepisLoader) for Resonite.
2. Download the latest release ZIP file (e.g., `GyztorMizirath-ResoniteInputControl-X.zip`) from the [Releases](https://github.com/Gyztor/ResoniteInputControl/releases) page.
3. Extract the ZIP and copy the `plugins` folder to your BepInEx folder in your Resonite installation directory:
   - **Default location:** `C:\Program Files (x86)\Steam\steamapps\common\Resonite\BepInEx\`
4. Start the game. If you want to verify that the mod is working you can check your BepInEx logs.
